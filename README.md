# GoodStack

A good webstack that focuses on modern tooling, type-safety, and shortcuts.

## Stack

| Asset     | Tool        |
| ---       | ---         |
| Document  | Pug         |
| Styles    | SCSS        |
| Scripting | Typescript  |
| Data      | Drizzle     |

## Optional Features

- Database client (MySQL)
- Rate limiting
